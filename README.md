# Green Rabbit Snack Hub — greenrabbit

Independent affiliate SEO site for Green Rabbit bulk office snack delivery via Warehouse115. **661 pages**, built and deployed in under 5 seconds.

**Live site:** https://brightlane.github.io/greenrabbit/

---

## Setup (5 minutes)

1. Upload `build.py` to the **root** of the `brightlane/greenrabbit` repo
2. Create `.github/workflows/deploy.yml` from the provided file
3. **Settings → Pages → Source → GitHub Actions**
4. Push to `main` — first build triggers automatically

No `ANTHROPIC_API_KEY` needed. No pip installs. Pure Python 3.

> **First run:** The workflow deletes every existing file in the repo. Only `build.py`, `README.md`, and `.github/` are preserved. The repo is rebuilt from scratch on every run.

- `SpeakableSpecification` — all pages (voice / AI assistant signal)
- `HowTo` with 5 named steps — how-to, guide, and buying pages
- `VideoObject` — how-to and guide pages (~46/50)
- `Review` with 4.7★ rating — review, comparison, general pages (~11/50)
- `ItemList` — product and category pages (~23/50)
- `Service` with `areaServed` — geo state and city pages
- `Organization` with `knowsAbout` — homepage
- `SearchAction` (SiteLinksSearchBox) — homepage
- `BlogPosting` + `Speakable` + `mentions` (Green Rabbit, Warehouse115) — blog posts

## Daily Rebuild

GitHub Actions rebuilds the entire 661-page site daily at **08:00 UTC**:

1. Deletes all old files (preserves only `build.py`, `README.md`, `.github/`)
2. Commits the clean state
3. Runs `python build.py` (~1–2 seconds)
4. Deploys fresh `output/` to GitHub Pages

Staggered vs the other two sites: onlinemovavi 06:00, hrcp 07:00, greenrabbit 08:00.
